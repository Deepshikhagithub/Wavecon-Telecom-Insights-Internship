## Dataset Overview:

This dataset comprises six CSV files, each containing valuable information related to AtliQo's 5G services and the telecommunication industry. Let's briefly introduce each of these files and their key columns:

**dim_cities:**

‚óècity_code: Unique city code.

‚óècity_name: Name of the city.

**dim_date:**

‚óèdate: Starting date of each month.

‚óèmonth_name: Abbreviated month names (Jan, Feb, Mar, etc.).

‚óèbefore/after_5g: Categories representing "Before 5G" and "After 5G" periods.

‚óètime_period: Unique sequence number for comparing months before and after 5G implementation.


**dim_plan:**

‚óèplan: Various internet plans offered by Wavecon.

‚óèplan_description: Brief description of the internet plan.

**fact_wavecon_metrics:**

‚óèdate: Starting date of each month.

‚óècity_code: Unique pincode code for each city.

‚óècompany: The company name (Wavecon).

‚óèatliqo_revenue_crores: Revenue generated by Wavecon in crores (1 Crore = 10 Million).

‚óèarpu: Average Revenue Per User (ARPU).

‚óèactive_users_lakhs: Number of active users in lakhs (1 Lakh = 100,000).

‚óèunsubscribed_users_lakhs: Number of unsubscribed users in lakhs (1 Lakh = 100,000).

**fact_market_share:**

‚óèdate: Starting date of each month.

‚óècity_code: Unique city code.

‚óètmv_city_crores: Total market value of the city in crores.

‚óècompany: Competitor names in the telecom industry (Wavecon, Britel, DADAFONE, PIO, Others).

‚óèms_pct: Percentage of market share gained by each company.

**fact_plan_revenue:**

‚óèdate: Starting date of each month.

‚óècity_code: Unique city code.

‚óèplans: Various internet plans offered by Wavecon.

‚óèplan_revenue_crores: Revenue generated by Wavecon from each plan in crores.

This dataset provides a comprehensive view of key metrics related to Wavecon's 5G services, including revenue, user data, market share, and plan-specific revenue. It serves as the foundation for conducting in-depth analyses and generating valuable insights to support data-driven decisions.
